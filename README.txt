SAP Joule Video Course Web Page

How to use:
1. Unzip the package.
2. Open index.html in a browser to preview.
3. For best results, upload the full folder to GitHub Pages, Netlify, GoDaddy, or any static hosting.
4. Replace assets/video/sap_joule_sample_course_video.mp4 with your final MP4 when ready.
5. Replace assets/audio/sap_joule_intro_narration.mp3 with your final professional narration.

Included:
- index.html: main course page
- styles.css: professional styling
- app.js: interactive slide viewer and searchable storyboard
- assets/slides: rendered PowerPoint slide images
- assets/video: MP4 sample generated from slides and narration
- assets/audio: sample MP3 narration
- assets/downloads: original PowerPoint, trainer script, storyboard, and CSV
